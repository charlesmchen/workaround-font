Influences

The primary influences on Workaround are:

  • VAG Rounded,
  • Stag Sans and
  • Ubuntu Title / Ubuntu Titling.

Other influences include:

  • Arial Rounded
  • Bryant
  • Cennerik
  • Chevin
  • FF DIN Round
  • FF Speak Web
  • Frankfurter
  • GE Inspira
  • Glow Gothic Overlay
  • Gotham Round
  • Helvetica Rounded
  • Hiruko
  • Maagkramp
  • MG Open Modata
  • Normalise DIN
  • Predicate Rounded
  • Report
  • Scheme
  • Simpliciter
  • TWC Round
  • Variable Black
